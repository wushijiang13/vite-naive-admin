## 现在支持的功能是：  
  多布局切换、tab展示页面、tab页快捷操作组件、tab页右键菜单、概览、图标、综合表格、行内编辑表格、列表、描述、日历、综合表单、分步表单、按钮、
  文字链接、单选框、多选框、输入框、计数器、选择器、开关、滑块、时间选择器、日期选择器、日期时间选择器、评分、图表、打印、通知、时间线、
  多标签、水印、分享、动态锚点、动态meta
